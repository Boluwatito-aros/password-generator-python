# Password Generator

A customisable password generator built in Python that produces cryptographically secure passwords.

## View the code
The full code is in `password_generator.py` above.

## About
Developed a customisable password generator in Python using the secrets and string modules
to produce cryptographically secure passwords of user-defined length and complexity.

## Features
- Generates passwords between 12 and 16 characters
- Guaranteed inclusion of uppercase, lowercase, digits, and special characters
- Configurable character sets giving users fine-grained control over password composition
- Password strength evaluator that scores generated passwords against common security criteria
  (length, entropy, character diversity) and provides feedback to the user
- Ensures generated password does not match the username
